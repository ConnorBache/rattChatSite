RattChat Plugin Documentation

Roleplay-oriented chat system with proximity, modes, and configurable formatting.

1. Overview

RattChat is a roleplay-focused chat plugin for Minecraft servers. It introduces multiple chat modes with configurable ranges and colors, enabling immersive local, indoor, and long-distance roleplay communication.

1.1 Chat modes

  • Whisper – Private, short-range communication
  • Indoor – Medium-range chat for in-building conversations
  • RP – Default roleplay mode for descriptive in-game narration
  • Yell – Long-range shouting across large areas
  • DM – Server-wide announcements; requires dedicated permission

1.2 Features

  • Configurable chat prefixes, colors, dialogue colors, and ranges via RattChatConfig.yml
  • Proximity-based chat visibility per mode
  • Automatic highlighting of dialogue enclosed in quotation marks (" ")
  • Permission-based restriction of DM mode

2. Installation

2.1 Install the plugin

Download or compile RattChat-1.0-SNAPSHOT.jar and place it into your server’s plugins directory: 

C:\Path\To\MinecraftServer\plugins\

2.2 Restart the server

Stop the server, move the jar if necessary, then start the server again. For example: 

stop

2.3 Verify that RattChat is loaded

Run the following command in the server console or in-game (with permissions):

/plugins

If RattChat is listed in green, the plugin has loaded successfully.

3. Commands

3.1 Chat mode commands

Command Description Permission
/chatmode Whisper Switches the player to Whisper mode (short range). rattchat.use
/chatmode Indoor Switches the player to Indoor mode (medium range). rattchat.use
/chatmode RP Switches the player to RP mode (default roleplay mode). rattchat.use
/chatmode Yell Switches the player to Yell mode (long range). rattchat.use
/chatmode DM Switches the player to DM mode (server-wide chat). rattchat.dm

4. Permissions

Permission Description Default
rattchat.use Allows players to switch between Whisper, Indoor, RP, and Yell modes. Everyone
rattchat.dm Allows use of DM mode (server-wide messages). Operators

5. Configuration

RattChat creates a configuration file on first load: 

/plugins/RattChatConfig.yml

Use this file to adjust mode prefixes, colors, dialogue colors, and proximity ranges.

5.1 Default configuration

ChatModes:
  Whisper:
    Prefix: "~Whisper~ "
    Color: LIGHT_PURPLE
    DialogueColor: DARK_PURPLE
    Range: 5

  Indoor:
    Prefix: "~Indoor~ "
    Color: GREEN
    DialogueColor: DARK_GREEN
    Range: 15

  RP:
    Prefix: "~RP~ "
    Color: YELLOW
    DialogueColor: GOLD
    Range: 30

  Yell:
    Prefix: "~Yell~ "
    Color: BLUE
    DialogueColor: DARK_BLUE
    Range: 50

  DM:
    Prefix: "~DM~ "
    Color: RED
    DialogueColor: DARK_RED
    Range: -1  # -1 = no distance limit

5.2 Editing the configuration

  1. Open RattChatConfig.yml in a text editor.
  2. Change any Prefix, Color, DialogueColor, or Range values.
  3. Save the file.
  4. Restart the server for changes to take effect.

Use standard Bukkit/Spigot color names for Color and DialogueColor.

6. In-game chat formatting

6.1 Mode message examples

Chat mode Example output
Whisper ~Whisper~ Hello there!
Indoor ~Indoor~ Welcome inside!
RP ~RP~ The knight approached.
Yell ~Yell~ HEY, EVERYONE!
DM ~DM~ This is a server announcement.

6.2 Dialogue highlighting

Any text enclosed in quotation marks is rendered using the configured dialogue color for the active mode. 

~RP~ The wizard said, "You must go now!"
